OVERVIEW

This program can remap river network data from spherical grid onto another spherical grid.

The program expects to read river network data from a netcdf file, which is specified by the namelist variable "river_input_file". This file should contains field 'cellarea', 'tocell', 'fromcell', 'basin', 'basincells', 'order', 'travel', 'subL', 'subA','disttomouth', 'disttoocean' and 'dx'. The program will remap the data in "river_input_file" onto the grid, which is specified by namelist land_grid_file. The grid file should contains the same grid as the land modle to be run. The output is stored in a netcdf file, which is specified by river_output_file.
