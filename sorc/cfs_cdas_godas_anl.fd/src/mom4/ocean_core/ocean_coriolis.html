OVERVIEW

Coriolis acceleration

This module computes Coriolis acceleration on a B-grid.

PUBLIC DATA

None.

PUBLIC ROUTINES

  1. ocean_coriolis_init

    DESCRIPTION
    Initialize the Coriolis module


  2. coriolis_field

    DESCRIPTION
    Compute thickness weighted acceleration due to Coriolis term on a B-grid


  3. coriolis_slice

    DESCRIPTION
    Compute acceleration due to Coriolis term on a B-grid for a particular depth level and particular direction.
